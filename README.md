# Partyhelp

Partyhelp Pay-Per-Lead management app — Laravel 12 + Filament 5 backend for the venue-lead marketplace.

## Documentation

- [PRD](/docs/PRD_Partyhelp_Pay_Per_Lead.md) — Product Requirements Document

## Tech Stack

- Laravel 12, Filament 5, MySQL
- Admin panel: `/admin`
- Venue portal: `/venue`
- Webhook: `POST /api/webhook/elementor-lead`
